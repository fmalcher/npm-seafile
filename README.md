npm module for accessing the [Seafile Web API](http://manual.seafile.com/develop/web_api.html).

It just implements the following subset of the available functions:
* `listAccounts`
* `createAccount`
* `updateAccount`
* `deleteAccount`
* `getAccountInfo`
* `addGroupMember`
* `deleteGroupMember`
* ...more coming soon!
